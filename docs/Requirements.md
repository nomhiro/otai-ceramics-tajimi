## 3. 技術的問題点

| 問題点 | 現状 | 影響・対応 |
|--------|------|------------|
| 文字コード | Shift_JIS（文字化け多発） | UTF-8に移行必須 |
| SSL/HTTPS | 非対応（http://） | セキュリティ警告、SEO低下 |
| レスポンシブ | 非対応（テーブルレイアウト） | スマホで閲覧困難 |
| HTML/CSS | 1990年代仕様（.htm拡張子） | 保守困難、SEO非対応 |
| CGIスクリプト | 検索・掲示板がCGI依存 | セキュリティリスク、廃止検討 |
| Copyright表記 | 1999-2009で停止 | サイト放置の印象 |
| 階層構造 | 深いネスト（3階層以上） | URL構造の整理が必要 |

## 4. EC機能の現状

### 4.1 販売条件

| 項目 | 内容 |
|------|------|
| 価格 | 税込表示、すべて手描きの一点もの |
| 最小注文 | 1個から注文可能 |
| 支払方法 | 代引、銀行振込、郵便振替 |
| 送料 | 地域別（740円〜1,260円）、10,000円以上で無料 |
| 納期 | 在庫品: 2〜3営業日、受注生産: 約25日 |
| 返品 | 未使用品に限り7日以内（破損・不良は全額返金） |

### 4.2 OEM/記念品対応

- オリジナルマグカップ、湯呑: 200円〜（版代別）
- 最小ロット: 1個から対応可能
- 大量注文: 1万円以上で別途割引あり
- 前払い制（オリジナル製品）

## 8. 非機能要件

### 8.1 パフォーマンス

- ページ読み込み時間: 3秒以内
- Core Web Vitals基準達成
- 画像最適化（WebP対応、遅延読み込み）

### 8.2 セキュリティ

- SSL/TLS暗号化
- フォームのCSRF対策
- 定期的なセキュリティアップデート

### 8.3 SEO対策

- 構造化データ（Schema.org）対応
- 適切なメタタグ設定
- サイトマップ（XML/HTML）
- パンくずリスト
- alt属性の適切な設定

### 8.4 アクセシビリティ

- WCAG 2.1 AA準拠
- キーボード操作対応
- スクリーンリーダー対応
